# 🎨 Portfolio - Angelica Gomez

Vollständiges Portfolio & CV mit allen Projekten, gestaltet nach dem **unwritten Styleguide**.

## 🎯 Features

- **Responsive Design** - Optimiert für Desktop, Tablet und Mobilgeräte
- **Projekt-Showcase** - 12+ Projekte in verschiedenen Kategorien
- **Filter-System** - Filterung nach Projekt-Typ (Games, Web, AI, Systeme)
- **unwritten Styleguide** - Farben, Fonts und Design nach Vorgabe
- **Performance** - Built mit Next.js für optimale Speed
- **Modern Stack** - React, TypeScript, Tailwind CSS

## 🚀 Installation & Nutzung

```bash
# Dependencies installieren
npm install

# Development Server starten
npm run dev

# Build für Produktion
npm build

# Production Server starten
npm start
```

Die App läuft dann auf `http://localhost:3000`

## 📦 Dependencies

- **Next.js 14** - React Framework
- **React 18** - UI Library
- **TypeScript** - Type Safety
- **Tailwind CSS** - Utility-first CSS
- **Octokit** - GitHub API (optional)

## 🔧 Konfiguration
