# 🛩️ FLY - Cessna 172 World Flight Trainer v13

**Eject Hero + World Landmarks Edition**

---

## 📋 Über dieses Spiel

Ein realistischer Flight Trainer basierend auf der Cessna 172. Mit:
- ✈️ Realistische Flugphysik
- 🌍 World Landmarks & Sehenswürdigkeiten
- 📍 GPS Navigation
- 🎮 Keyboard Controls
- 📊 Instrument Panel (Höhe, Geschwindigkeit, Kompass)

---

## 🚀 Quick Start

### Installation
```bash
npm install
```

### Server starten
```bash
npm start
```

### Browser öffnen
```
http://localhost:3001
```

---

## 🎮 Controls

| Taste | Funktion |
|-------|----------|
| **W/A/S/D** | Fliegen (nach oben/links/unten/rechts) |
| **Q/E** | Roll (Links/Rechts drehen) |
| **Space** | Höhe halten / Autopilot |
| **Shift** | Schneller fliegen |
| **M** | Landkarte anzeigen/verbergen |
| **N** | Navigationsmarker setzen |
| **L** | Landmark suchen & anfliegen |
