## 🚀 So öffnest du es in PyCharm

1. **PyCharm starten**
2. **File → Open**
3. **Wähle das Verzeichnis**: `C:\Users\Anjel\Desktop\Work Projekt\python coden`
4. **"Open as Project"** klicken
5. Fertig! Du kannst jetzt jedes Projekt einzeln öffnen und bearbeiten

## 💡 So benutzt du die Projekte

1. **Öffne einen Projektordner** z.B. `01_calculator`
2. **Öffne `main.py`** in PyCharm
3. **Lies die `README.md`** für Erklärungen
4. **Führe das Programm aus**: `python main.py` (oder Ctrl+Shift+F10 in PyCharm)
5. **Experimentiere & ändere Code**
6. **Lerne die Konzepte** - jedes Projekt lehrt etwas anderes

## 🎯 Lernziele pro Projekt

- **01-06**: Grundlagen (Input/Output, If-Else, Schleifen)
- **07-12**: Datenstrukturen (Listen, Dictionaries, Funktionen)
- **13-15**: Text & Mathematik (String-Methoden, datetime)
- **16-20**: Komplexere Logik (Spielmechaniken, Patterns, Exceptions)

- ✅ Listen & Dictionaries
- ✅ Funktionen

- ✅ String-Methoden
- ✅ Mathematische Operationen
