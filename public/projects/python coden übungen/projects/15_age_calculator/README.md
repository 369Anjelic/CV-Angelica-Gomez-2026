# 🎂 Altersrechner

Berechnet Alter und Lebenstage.

## Features
- Berechnet aktuelles Alter
- Berücksichtigt ob Geburtstag schon war
- Zeigt gelebte Tage
- `datetime` Modul

## Wie man es ausführt
```bash
python main.py
```

## Verbesserungen möglich
- Nächster Geburtstag
- Alter in Stunden/Minuten
- Astrologische Zeichen

## Was du lernen kannst
- `datetime` Modul
- Zeitberechnung
- Bedingte Logik
- Datum-Arithmetik
