# 🔐 Passwort Generator

Erstelle sichere, zufällige Passwörter.

## Features
- Anpassbare Länge
- Buchstaben, Zahlen, Sonderzeichen
- Sichere Zufallsgenerierung
- Validierung der Länge

## Wie man es ausführt
```bash
python main.py
```

## Was du lernen kannst
- `string` Modul
- List Comprehension
- `random.choice()`
- Sicherheit in Code
