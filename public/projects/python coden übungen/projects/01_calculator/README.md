# 🧮 Taschenrechner

Ein einfacher Taschenrechner für die vier Grundrechenarten.

## Features
- Addition (+)
- Subtraktion (-)
- Multiplikation (*)
- Division (/)
- Fehlerbehandlung für Division durch Null

## Wie man es ausführt
```bash
python main.py
```

## Was du lernen kannst
- Benutzer-Input mit `input()`
- If-elif-else Struktur
- Float-Konvertierung
- String-Formatierung
