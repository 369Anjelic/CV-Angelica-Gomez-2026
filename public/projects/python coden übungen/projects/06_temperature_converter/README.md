# 🌡️ Temperaturkonverter

Konvertiert Temperaturen zwischen verschiedenen Einheiten.

## Features
- Celsius ↔ Fahrenheit
- Celsius ↔ Kelvin
- Mathematische Formeln
- Menü-Auswahl

## Wie man es ausführt
```bash
python main.py
```

## Was du lernen kannst
- Mathematische Operationen
- Menü-basierte Benutzer-Schnittstelle
- Float-Arithmetic
- Temperaturformeln
