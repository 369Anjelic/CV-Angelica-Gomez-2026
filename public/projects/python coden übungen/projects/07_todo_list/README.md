# ✅ To-Do Liste

Eine einfache Aufgabenverwaltungs-App mit Persistierung.

## Features
- Aufgaben hinzufügen
- Aufgaben anzeigen
- Aufgaben abhaken
- Menü-System

## Wie man es ausführt
```bash
python main.py
```

## Verbesserungen möglich
- Aufgaben in Datei speichern
- Aufgaben löschen
- Prioritäten setzen

## Was du lernen kannst
- Listen-Verwaltung
- Dictionaries für Datenstruktur
- While-Schleifen
- Daten-Strukturen
