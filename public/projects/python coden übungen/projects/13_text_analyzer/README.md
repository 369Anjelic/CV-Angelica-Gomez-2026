# 📊 Text Analyzer

Analysiert Texte und zeigt Statistiken.

## Features
- Zeichenanzahl
- Wortanzahl
- Zeilenanzahl
- Eindeutige Wörter
- Durchschnittliche Wortlänge

## Wie man es ausführt
```bash
python main.py
```

## Verbesserungen möglich
- Häufigste Wörter
- Längster Satz
- Lesbarkeitsindex

## Was du lernen kannst
- String-Methoden
- Mathematik
- Text-Analyse
- Set für eindeutige Elemente
