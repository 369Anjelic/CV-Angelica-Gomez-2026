# 🔺 Pyramiden-Muster

Generiert verschiedene ASCII-Pyramiden.

## Features
- Sternen-Pyramide
- Zahlen-Pyramide
- Umgekehrte Pyramide
- Hohlpyramide (nur Ränder)
- Beliebige Größe

## Wie man es ausführt
```bash
python main.py
```

## Verbesserungen möglich
- Farbige Ausgabe
- Diamanten-Muster
- Andere Formen

## Was du lernen kannst
- Verschachtelte Schleifen
- String-Manipulation
- ASCII-Art
- Mathematische Muster
