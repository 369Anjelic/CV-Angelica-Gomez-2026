# 💭 Zufälliger Zitat Generator

Zeige motivierende Zitate an.

## Features
- Liste von Zitaten
- Zufällige Auswahl
- Wiederholbar
- Exit-Optionen

## Wie man es ausführt
```bash
python main.py
```

## Verbesserungen möglich
- Zitate aus Datei laden
- Nach Kategorie filtern
- Zitate speichern

## Was du lernen kannst
- Listen verwenden
- `random.choice()`
- While-Schleifen
- Benutzereingabe verarbeiten
