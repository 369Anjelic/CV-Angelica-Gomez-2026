# ⚖️ BMI Rechner

Berechnet deinen Body Mass Index und bewertet ihn.

## Features
- Berechnet BMI aus Gewicht/Größe
- Kategorisierung (Untergewicht bis Adipositas)
- Einfache Eingabe
- Aussagekräftige Ausgabe

## Wie man es ausführt
```bash
python main.py
```

## BMI-Kategorien
- Untergewicht: < 18.5
- Normalgewicht: 18.5 - 25
- Übergewicht: 25 - 30
- Adipositas: > 30

- Mathematische Formeln
- Bedingte Logik
