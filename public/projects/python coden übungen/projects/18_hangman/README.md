# 🎮 Galgenmännchen (Hangman)

Klassisches Galgenmännchen-Spiel.

## Features
- Zufälliges Wort auswählen
- Buchstaben raten
- Versuche zählen
- Fortschritt anzeigen
- Falsche Buchstaben merken

## Wie man es ausführt
```bash
python main.py
```

## Verbesserungen möglich
- Schwierigkeitsstufen
- Wort-Kategorien
- Spielhistorie
- Mehr Wörter hinzufügen

## Was du lernen kannst
- Spiellogik
- Listen-Manipulation
- String-Iteration
- Game Loop
