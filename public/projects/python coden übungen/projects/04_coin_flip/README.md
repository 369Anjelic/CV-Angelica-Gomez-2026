# 🪙 Münzwurf Simulator

Werfe eine virtuelle Münze und gewinne oder verliere!

## Features
- Zufällige Ergebnisse
- Benutzer-Vorhersage
- Schleife für mehrere Würfe
- Emoji-Ausgabe

## Wie man es ausführt
```bash
python main.py
```

## Was du lernen kannst
- `random.choice()`
- Schleifen mit Bedingungen
- String-Vergleiche
- Spielmechaniken
