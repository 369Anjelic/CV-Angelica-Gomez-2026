# ✂️ Schnick-Schnack-Schnuck

Spielen Sie Schnick-Schnack-Schnuck gegen den Computer!

## Features
- KI-Computer-Gegner
- Benutzer-freundliche Ausgabe
- Endlosschleife mit Quit-Option
- Logik für Sieg/Niederlage/Unentschieden

## Wie man es ausführt
```bash
python main.py
```

## Was du lernen kannst
- Komplexe If-Bedingungen mit AND/OR
- Spiellogik mit mehreren Bedingungen
- String-Verarbeitung
- Zufallsauswahl
