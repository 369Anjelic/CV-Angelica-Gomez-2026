# 🎯 Zahlenratespiel

Der Computer denkt sich eine Zahl aus, und du musst sie erraten!

## Features
- Zufallszahlengenerierung
- Hinweise (höher/niedriger)
- Zählt die Versuche
- Try-except für Fehlerbehandlung

## Wie man es ausführt
```bash
python main.py
```

## Was du lernen kannst
- `random` Modul
- While-Schleifen
- Try-except Exception Handling
- Spiellogik programmieren
