# 🎓 Quiz

Ein einfaches Quiz mit Scoring-System.

## Features
- Mehrere Fragen
- Automatische Auswertung
- Punktesystem
- Richtige Antwort anzeigen

## Wie man es ausführt
```bash
python main.py
```

## Verbesserungen möglich
- Mehr Fragen hinzufügen
- Schwierigkeitsstufen
- Fragen aus Datei laden

## Was du lernen kannst
- Listen mit Dictionaries
- Enumerate-Funktion
- Scoring-Logik
- Datenstruktur für Fragen
