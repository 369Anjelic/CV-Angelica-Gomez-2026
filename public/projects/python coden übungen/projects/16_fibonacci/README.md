# 🔢 Fibonacci Generator

Generiert Fibonacci-Sequenzen mit Visualisierung.

## Features
- Beliebig viele Fibonacci-Zahlen
- ASCII-Balken-Visualisierung
- Einfache Logik mit Tuple-Unpacking
- Liste ausgeben

## Wie man es ausführt
```bash
python main.py
```

## Verbesserungen möglich
- Bestimmte Fibonacci-Zahl finden
- Grafische Visualisierung
- Goldener Schnitt
