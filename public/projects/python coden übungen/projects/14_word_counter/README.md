# 🔤 Wort-Zähler

Zählt, wie oft ein Wort in einem Text vorkommt.

## Features
- Wort-Häufigkeit zählen
- Top 5 häufigste Wörter anzeigen
- Case-insensitive
- `collections.Counter` Modul

## Wie man es ausführt
```bash
python main.py
```

## Verbesserungen möglich
- Stopwords filtern
- Häufigkeitsdiagramm
- Mehrere Wörter auf einmal

## Was du lernen kannst
- `collections` Modul
- `Counter` Klasse
- `most_common()` Methode
- Text-Analyse
