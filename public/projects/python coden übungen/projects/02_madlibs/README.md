# 📖 Mad Libs

Ein interaktives Spiel, bei dem der Spieler Wörter eingeben muss, um eine lustige Geschichte zu erstellen.

## Features
- Interaktive Eingabe
- String-Interpolation
- Zufällige Geschichten möglich

## Wie man es ausführt
```bash
python main.py
```

## Was du lernen kannst
- f-Strings für Textformatierung
- Listen (erweiterbar für mehrere Geschichten)
- User Input verarbeiten
