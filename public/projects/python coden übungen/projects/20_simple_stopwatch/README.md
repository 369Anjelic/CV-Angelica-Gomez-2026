# ⏱️ Einfache Stoppuhr

Eine Stoppuhr mit Countdown und Zeitmesser.

## Features
- Countdown von X Sekunden
- Zeitmesser (Ctrl+C zum Stoppen)
- Echtzeit-Anzeige
- `time` Modul

## Wie man es ausführt
```bash
python main.py
```

## Verbesserungen möglich
- Minute/Sekunden-Format
- Mehrere Timer
- Speichern von Zeiten
- Alarm-Funktion

## Was du lernen kannst
- `time` Modul
- `time.sleep()`
- `time.time()`
- Real-Time Output mit flush
- Exception Handling (KeyboardInterrupt)
