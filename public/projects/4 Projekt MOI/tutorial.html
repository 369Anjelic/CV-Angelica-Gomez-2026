🤖 Bot selber bauen - Anfänger-Anleitung

Schritt für Schritt: Von Null zum funktionierenden Bot in unter 30 Minuten!

1 Was du brauchst (5 min)

✅ Voraussetzungen

Du brauchst nur 3 Dinge:

Claude Abo - Kostenlos mit grundlegem Plan von Claude.ai oder Bezahlplan
Einen Code-Editor - VS Code (kostenlos, empfohlen)
Node.js - Damit dein Bot laufen kann

Installation - Schritt für Schritt

Schritt 1a: Claude Abo

  1. Gehe zu claude.ai
  2. Klick auf "Sign Up" (kostenlos)
  3. Gib deine Email ein
  4. Bestätige die Email
  5. Fertig! ✓ Du hast jetzt Zugriff auf Claude KI
💡 Tipp: Der kostenlose Plan ist für Anfänger perfekt. Du kannst später upgraden wenn du mehr brauchst.

Schritt 1b: VS Code herunterladen

  1. Gehe zu code.visualstudio.com
  2. Klick auf den großen grünen Download-Button
  3. Installiere VS Code (wie jedes andere Programm)
  4. Starten und fertig! ✓

Schritt 1c: Node.js installieren

  1. Gehe zu nodejs.org
  2. Klick auf "LTS" (langfristig unterstützt, sicherer)
  3. Installiere es (Standard-Einstellungen sind okay)
  4. Öffne Terminal/PowerShell und gib ein:
node --version

Wenn eine Versionsnummer erscheint (z.B. v20.0.0), ist alles OK! ✓

✅ Fertig mit Schritt 1! Du hast alles, was du brauchst. Weiter zu Schritt 2!

2 Das MOI Bot-Projekt (5 min)

📁 Projekt herunterladen oder klonen

Es gibt zwei Möglichkeiten:

Option A: Einfach (empfohlen für Anfänger)

  1. Gehe zu diesem MOI Bot Projekt
  2. Klick auf Code → Download ZIP
  3. Entpack die ZIP-Datei in einen Ordner deiner Wahl
  4. Öffne den Ordner in VS Code

Option B: Mit Git (fortgeschrittener)

git clone https://github.com/unwritten-studio/moi-bot.git
cd moi-bot
code .
💡 Tipp: Wenn du Git nicht kennst, nimm Option A. Das ist völlig ausreichend!

📦 Abhängigkeiten installieren

Der Bot braucht ein paar Programmteile ("Dependencies"). Die installierst du so:

  1. Öffne das Terminal in VS Code: Ctrl + Backtick (unten)
  2. Tippe ein:
npm install

Das kann 1-2 Minuten dauern. Das Terminal wird sagen wenn es fertig ist.

✅ Fertig mit Schritt 2! Der Bot ist jetzt bereit zum Starten.

3 API Key einrichten (3 min)

🔑 Claude API Key besorgen

Der Bot muss wissen, dass ER dein Bot ist. Dafür brauchst du einen "API Key":

  1. Gehe zu console.anthropic.com
  2. Meld dich an (mit deinem Claude-Account)
  3. Klick auf "API Keys" oder "Keys"
  4. Klick auf "Create Key"
  5. Gib ihm einen Namen (z.B. "MOI Bot")
  6. Klick "Create"
  7. Kopiere den Key (beginnt mit sk-ant-)
  8. WICHTIG: Zeige diesen Key NIEMANDEM! Speichere ihn sicher!
⚠️ SICHERHEIT WICHTIG: Dieser Key ist wie ein Passwort. Wenn jemand ihn hat, kann er deinen BOT missbrauchen. Niemals öffentlich teilen oder in GitHub pushen!

.env Datei einrichten

Der Key kommt in eine ".env" Datei:

  1. Öffne VS Code und gehe in den MOI-Ordner
  2. Klick auf .env.example Datei
  3. Kopiere den Inhalt (Ctrl+A, Ctrl+C)
  4. Neue Datei: Ctrl+N
  5. Einfügen: Ctrl+V
  6. Speichern als ".env" (nicht .env.example!)
  7. Suche nach ANTHROPIC_API_KEY=
  8. Ersetze "xxx" mit deinem API Key (den du gerade kopiert hast)
  9. Speichern: Ctrl+S
ANTHROPIC_API_KEY=sk-ant-dein-key-hier-einfuegen
✅ Fertig mit Schritt 3! Dein Bot weiß jetzt, wer er ist.

4 Erstes Starten (2 min)

🚀 Server starten

Jetzt lässt du deinen Bot laufen:

  1. Terminal in VS Code (Ctrl + Backtick)
  2. Tippe ein:
npm run server

Du solltest sehen:

🤖 ========================================
   MOI Bot Server v10.5.2026
🤖 ========================================

✅ Server läuft auf http://localhost:3000

💻 Bot im Browser öffnen

  1. Öffne deinen Browser (Chrome, Firefox, Edge)
  2. Gehe zu: http://localhost:3000
  3. Du siehst jetzt die MOI Bot Homepage!
💡 Tipp: "localhost:3000" bedeutet "dieser Computer, Port 3000". Nur du kannst das sehen. Das ist wie ein lokales Testgebiet.

Teste deinen Bot!

  1. Schreib eine Frage: z.B. "Wie baue ich einen Bot?"
  2. Klick Send oder drück Enter
  3. MOI antwortet dir mit KI-Kraft! 🎉
✅ GLÜCKWUNSCH! Dein Bot läuft! Das war's schon!

5 Deinen Bot anpassen (10 min)

🎨 Erste Anpassungen

Jetzt machst du deinen eigenen Bot! Hier sind einfache Änderungen:

Änderung 1: Bot-Name

  1. Öffne die .env Datei
  2. Suche: BOT_NAME=MOI
  3. Ändere zu deinem Namen, z.B.: BOT_NAME=Mein-Super-Bot
  4. Speichern
  5. Server neustarten: Terminal, Ctrl+C zum Stoppen, dann "npm run server" wieder

Änderung 2: Homepage-Text

  1. Öffne index.html
  2. Suche nach "MOI - Dein AI-Bot-Experte"
  3. Ändere den Text zu deinem Titel
  4. Speichern, Browser aktualisieren (F5)

Änderung 3: Bot-Anweisungen

Der Bot wird durch sein "System Prompt" gesteuert:

  1. Öffne bot.js
  2. Suche: const SYSTEM_PROMPT =
  3. Ändere die Anweisungen, z.B.:
const SYSTEM_PROMPT = `Du bist mein persönlicher AI-Assistent.
- Du hilfst mir bei Programmier-Fragen
- Du erklärst alles verständlich
- Du gibst Tipps zum Programmieren
- Du bist freundlich und hilfreich`;

Speichern, Server neustarten, fertig!

💡 Tipp: Der System Prompt ist der Charakter deines Bots. Hier kannst du ihm sagen, wie er sich verhalten soll!

Änderung 4: Farben ändern

Die MOI Farben sind: #3987b8 (Blau) und #3e4447 (Dunkelgrau)

  1. In der index.html oder bot.js
  2. Suche: #3987b8
  3. Ersetze mit deiner Lieblingsfarbe (z.B. #FF5733 für Orange)
  4. Browser aktualisieren (F5)
✅ Fertig mit Schritt 5! Dein Bot ist jetzt personalisiert!

6 Online stellen (Deployment) (5 min)

🌍 Bot ins Internet bringen

Dein Bot läuft jetzt nur lokal. Um ihn mit der Welt zu teilen, musst du ihn "deployen":

Einfachste Methode: Vercel (EMPFOHLEN)

  1. Gehe zu vercel.com
  2. Klick "Sign Up" (mit GitHub/Google/Email)
  3. Klick "Import Project"
  4. Paste den Link zu deinem Bot-Projekt (GitHub)
  5. Vercel fragt nach Environment Variables
  6. Trage ein: ANTHROPIC_API_KEY = dein API Key
  7. Klick "Deploy"
  8. Nach 1-2 Minuten: Dein Bot hat eine eigene URL!
💡 Voraussetzung: Dein Projekt muss auf GitHub sein. Das ist wie OneDrive aber für Code. Kostenlos!

Alternative: Railway

  1. Gehe zu railway.app
  2. Meld dich an
  3. Klick "New Project"
  4. Verbinde dein GitHub-Repo
  5. Fertig! Railway macht den Rest automatisch
⚠️ WICHTIG: Vergiss nicht, deine Umgebungsvariablen (API Key) auf dem Hosting-Service einzubauen! Sonst funktioniert dein Bot nicht.
✅ Fertig mit Schritt 6! Dein Bot ist jetzt online! 🎉
