# 🔤 Palindrom Prüfer

Prüft ob Wörter oder Sätze Palindrome sind.

## Features
- Prüft Palindrome (vorwärts = rückwärts)
- Case-insensitive
- Ignoriert Leerzeichen
- Beispiele eingebaut

## Wie man es ausführt
```bash
python main.py
```

## Beispiele
- "Racecar" ✅
- "Level" ✅
- "Hello" ❌

## Verbesserungen möglich
- Nur Buchstaben (keine Zahlen)
- Sätze mit Sonderzeichen
- Mehrere Sprachen

## Was du lernen kannst
- String Reversal ([::-1])
- String-Methoden
- If-Bedingungen
- Logische Vergleiche
