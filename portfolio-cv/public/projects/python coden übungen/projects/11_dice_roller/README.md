# 🎲 Würfel Roller

Simuliere einen oder mehrere Würfelwürfe.

## Features
- Einzelner Würfelwurf (1-6)
- Mehrere Würfel werfen
- Summe berechnen
- Endlosschleife mit Menü

## Wie man es ausführt
```bash
python main.py
```

## Verbesserungen möglich
- D20, D100 Würfel
- Würfelhistorie
- Statistiken

## Was du lernen kannst
- List Comprehension
- `random.randint()`
- `sum()` Funktion
- Spielmechaniken
