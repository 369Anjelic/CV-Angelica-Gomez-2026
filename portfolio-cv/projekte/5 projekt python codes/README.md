**Umfangreiche Sammlung von 20 Python-Programmierübungen für Anfänger bis Fortgeschrittene**

## 📋 Über das Projekt

Eine strukturierte Sammlung von 20 verschiedenen Python-Übungen, die von einfachen Konzepten bis zu fortgeschrittenen Algorithmien reichen. Perfekt zum Lernen und Üben von Python-Programmierung.

**Status:** ✅ Aktiv & Vollständig  
**Version:** 2.0  
**Letztes Update:** 04.06.2026

## 🐛 Fehlerbehandlung

Falls der Server nicht startet:

```powershell
# 1. Alle Python-Prozesse killen
Get-Process python | Stop-Process -Force

# 2. Neu starten
python debug_serve.py

# 3. Port prüfen
netstat -an | findstr :5000
```
