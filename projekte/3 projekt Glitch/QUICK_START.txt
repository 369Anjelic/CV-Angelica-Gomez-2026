"Connection refused: localhost:3003"
→ Backend Server NOT running
→ Terminal 2: "npm start"

"API Key invalid"
→ .env Datei fehlt oder kaputt
→ Prüfe: ANTHROPIC_API_KEY=sk-ant-...

"Address already in use: :3000 oder :3003"
→ Port belegt von anderem Process
→ Ändere in index.html:1796 oder server.js:9
→ Oder: Kill anderen Process auf dem Port

Keine Datei wird erstellt
→ Schreib eine Nachricht im Chat
→ Dann guckt es in data/ Ordner
